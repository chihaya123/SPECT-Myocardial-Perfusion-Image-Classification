• Developed and compared two classification pipelines for SPECT myocardial perfusion imaging: a knowledge-based classifier 
using MATLAB image processing (segmentation, feature extraction, rule-based thresholding) and a deep learning classifier fine
tuning the pre-trained VGG16 network on perfusion image datasets. 

• Demonstrated that the deep learning approach significantly outperformed the rule-based system on sensitivity for ischemia 
detection while the knowledge-based system offered greater interpretability for borderline cases.
